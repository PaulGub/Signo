Signo, le générateur de signature mail du Groupe Bacqueyrisses

Choisissez si vous utilisez une adresse personnelle ou celle d'un service.

Saisissez votre prénom et votre nom. Par exemple : Paul Gubbiotti.

Indiquez votre poste (ou nom du département). Exemple : Assistant Marketing.

Entrez votre adresse mail professionnelle. Par exemple : exemple@bacqueyrisses.fr.

Sélectionnez la filiale du Groupe Bacqueyrisses à laquelle vous appartenez.

Uniquement si vous dépendez de la filiale Bacqueyrisses : choisissez la ville de votre site.

Indiquez votre numéro de ligne fixe si vous en possédez un. Exemple : 05 57 92 18 28.

Indiquez votre numéro de téléphone mobile si vous souhaitez qu’il apparaisse dans la signature. Exemple : 07 68 50 44 14.

Collez l’URL de votre photo de profil. Exemple : https://votrephoto.jpg
